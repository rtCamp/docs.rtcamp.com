Clean NGINX FastCGI cache, Opcache, Memcache

	ee clean

clean fastcgi cache by default.


### Clean FastCGI cache

	ee clean --fastcgi

### Clean Memcache

	ee clean --memcache

### Clean OPcache

	ee clean --opcache

### Clean Pagespeed cache

	ee clean --pagespeed

### Clean Redis cache

	ee clean --redis

### Clean FastCGI, Memcache, OPcache and PageSpeed cache

	ee clean --all
